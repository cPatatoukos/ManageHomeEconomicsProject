# Οικογενειακά Οικονομικά

Εφαρμογή διαχείρισης οικογενειακών εσόδων και εξόδων με Python, SQLite, tkinter, matplotlib και εξαγωγή Excel.

## Εγκατάσταση

```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Εκκίνηση

```bash
python main.py
```

Ή απευθείας:

```bash
python gui.py
```

## Δομή project

| Αρχείο | Ρόλος |
|--------|--------|
| `schema.sql` | Σχήμα βάσης (users, categories, transactions, recurring) |
| `database.py` | CRUD, μηνιαία επανάληψη, queries |
| `gui.py` | Γραφικό περιβάλλον (login, dashboard, φόρμες) |
| `analytics.py` | Γραφήματα matplotlib |
| `export.py` | Εξαγωγή μήνα σε `.xlsx` |
| `main.py` | Entry point |

## Χρήση

1. **Εγγραφή / Σύνδεση** — δημιουργήστε λογαριασμό ή συνδεθείτε.
2. **Αρχική** — σύνοψη εσόδων/εξόδων για επιλεγμένο μήνα.
3. **Συναλλαγές** — προσθήκη, επεξεργασία, διαγραφή. Επιλέξτε κατηγορία από λίστα ή «Νέα κατηγορία». Τσεκάρετε «Μηνιαία» για σταθερά ποσά που δημιουργούνται αυτόματα κάθε μήνα.
4. **Κατηγορίες** — διαχείριση κατηγοριών εσόδων και εξόδων.
5. **Μηνιαία** — διαχείριση σταθερών ποσών (μισθός, δόση δανείου).
6. **Στατιστικά** — pie, bar, line με ελεύθερο χρονικό εύρος και φίλτρο κατηγορίας.
7. **Εξαγωγή** — Excel (.xlsx) με στήλη χρήστη για οικογενειακή προβολή.

## Προβολή δεδομένων

Σε Αρχική, Συναλλαγές, Στατιστικά και Εξαγωγή μπορείτε να επιλέξετε:
- **Όλη η οικογένεια** — όλοι οι χρήστες
- **Μόνο εγώ** — μόνο ο συνδεδεμένος χρήστης

## Σημειώσεις

- Η βάση αποθηκεύεται στο `finance.db` (δημιουργείται αυτόματα).
- Κατά τη σύνδεση γίνεται συγχρονισμός μηνιαίων προτύπων για τον τρέχοντα μήνα.
